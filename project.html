PROJECTS

History of Typography

History of Typography is an animation project created with After Effects and Premiere. The animation is a quick overview of typographic evolution, illustrated with a metophor of human evolution.

The Visitor

The Visitor is a web-based comic story. Panels are drawn with Photoshop and animated with Javascript.

HoloGlass

HoloGlass is an application prototype developed on the Microsoft HoloLens for curators to review holographic art portfolios. HoloGlass is an UX design project born out of the Microsoft HoloLens Design Challenge at NYU Tandon.

Thumbs Up or Down

Thumbs Up or Down is a short book on good and bad design, created with InDesign.

Little Einstein

Little Einstein is a UX web design project for an online STEAM toy store. Via UX methods such as user research, card sorting, user personas, user flow, we built an Axure prototype.

Gill Sans

Gill Sans is a piece of type specimen design, created with Adobe Illustrator.

Fall for You

Animated lyrics of Fall for You by Secondhand Serenade, using After Effects. Project was created for the class Introduction to Digital Art.

Marcos El Oso

Marcos el Oso is a Spanish children's book, designed with InDesign. (Illustration credits to various artists online)
